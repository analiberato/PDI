# PDI
Processamento Digital de Imagens em SWT apresenta:

    + Filtros da média, mediana, vizinhança (estrela, cruz e quadrada)
    + Filtros morfológicos (erosão e dilatação)
    + Escala de cinza
    + Negativa
    + Segmentação
    + Rotação
    + Limiarização
    + Equalização
    + Histograma
    + Adição e subtração
    + Demarcar imagem
    + OpenCV
